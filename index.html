Graphist Gallery

Un léger script de galerie en PHP

 

Bienvenue sur le site web de Graphist Gallery !

Présentation

Graphist Gallery est un script de galerie multimédia dynamique ne demandant aucune base de données, votre contenu s'articulant autour d'une arborescence de dossier. Simple et sans fioriture, Graphist Gallery permet à l'utilisateur d'être plus concentré sur le contenu que sur le conteneur.

Fonctionnalités de Graphist Gallery

  • Afficher des photos et des vidéos classées dans une arborescence de dossier
  • Commenter tout un dossier ou une image individuelle
  • Gérer des pages statiques pour mettre en place un véritable site web
  • S'adapter très facilement à tout type de thème
  • Programme multilingue et facile à traduire
  • Et plus encore...

Comment installer Graphist Gallery

Pour installer Graphist Gallery, vous devez simplement disposer d'un petit espace web acceptant PHP et d'un accès, par exemple ftp, à ce dernier.

Etape 1

Téléchargez la dernière version de Graphist Gallery plus bas et extrayez le contenu de l'archive.

Etape 2

Connectez vous à votre espace web et envoyez-y le contenu de l'archive. Graphist Gallery est prêt à être utilisé.

Comment commencer à utiliser Graphist Gallery

Vous voici maintenant avec un Graphist Gallery fraichement envoyé sur le web et prêt à être utilisé.

Contenu

Le but de Graphist Gallery est de gérer localement votre dossier de contenu, hors ligne, pour l'envoyer ensuite directement sur votre espace web, via ftp ou autre connexion. Le contenu doit se trouver dans le dossier content/. Vous pouvez classer images et vidéos dans une arborescence à deux niveaux (dossiers et sous-dossier). Il vous suffit simplement d'ajouter, par exemple, une image dans un dossier vide pour qu'il soit affiché sur votre site. 

content/
|
|- Paysages/
|       |- image1.jpg
|       |- image2.jpg
|
|- Portraits/
|       |- Noir et Blanc
|                   |- image1.jpg

Pour ajouter une vidéo - format webm, mp4 ou ogv, sachant que vous pouvez mettre une même vidéo dans ces trois formats - il vous suffit la placer dans un dossier et de mettre le terme video dans le titre de cette dernière (par exemple : video-photocopieuse.webm). De plus, vous pouvez insérer des vidéos de sites tiers. Il vous suffit de créer un fichier sans extension avec le terme video dans son nom et de coller dans le fichier le code de partage de la vidéo.

Commentaires

Il y a deux sortes de commentaires : ceux qui englobent tout un dossier et les commentaires pour une image individuelle (ne fonctionne que pour les images pour le moment). Pour gérer cela, il suffit juste de vous connecter à l'espace d'administration puis de revenir sur votre site. Vous pouvez alors ajouter, modifier et supprimer tout type de commentaires comme bon vous semble.

Pages

Pour gérer les pages statiques et permettre à votre galerie de prendre des allures de site web complet, il vous suffit simplement de vous connecter à l'administration et de vous rendre dans la partie d'administration des pages.

Options avancées

Graphist Gallery ne s'arrête évidemment pas à cela. Vous pouvez créer de multiples dossiers de contenu, classer dans l'ordre que vous souhaitez vos dossiers, vos médias, afficher les informations exif d'une photo, adapter votre instance de Graphist Gallery à n'importe quel thème très facilement, personnaliser le nom de chaque dossiers... Je vous invite à lire la documentation pour prendre connaissance de tout cela.
